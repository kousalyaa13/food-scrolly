# 🥗 Think You Eat Healthy?

This project is a **scrollytelling website** that explores how food labels and assumptions about “healthy” foods can be misleading. It uses interactive visualizations to help users understand how calories, fat, and other nutrients vary across different foods. The website guides users through a story using data and visuals:

- Compare foods that look healthy vs. unhealthy  
- Explore relationships between calories and fat  
- See how nutrition differs across food categories  
- Understand the difference between **drinking fruit vs. eating fruit**  
- Discover how small choices can impact overall nutrition  

---

## Features

- Interactive scatter plot (Calories vs. Fat)  
- Dynamic bar chart (switch between nutrients)  
- Scrollytelling layout with smooth transitions  
- Visual comparisons (e.g., fruit vs. smoothie)  
- Custom cartoon-style illustrations  

---

## Tools Used

- HTML, CSS, JavaScript  
- Chart.js (data visualizations)  
- PapaParse (CSV data handling)
- Claude Code
